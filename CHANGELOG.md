# Changelog

All notable changes to DevHub extension will be documented here.

## [0.1.0] - 2025-11-04

### Added

- Initial release
- TreeView sidebar with 5 services (GitHub, MongoDB, Authentication, Stripe, LemonSqueezy)
- Hierarchical service display (Server → Details → Actions)
- 10 command palette commands
- Interactive webview dashboard
- Real-time status indicators
- Context menu actions (Connect, Disconnect, Configure, Test)
- Keyboard shortcuts for common actions
- MCP server integration foundation
- Event-driven architecture with EventEmitter
- VS Code theme integration (dark/light mode support)

### Features

- Connect/Disconnect to services
- Test connection health
- View service configuration
- Refresh service list
- Activity logging
- Server statistics
- Batch operations (Connect All, Disconnect All)

### Technical

- TypeScript with strict typing
- VS Code Extension API integration
- MCP SDK implementation
- Modular architecture
- Error handling and recovery
- State management
- WebView communication
